## Decisions & Tradeoffs
- **Internal Python package kept as `curriculum_gen`**: The import path (`from curriculum_gen.cli import app`) does not need to match the project name. Renaming the package directory would create unnecessary churn without user-facing benefit.
- **`project.md` split into architecture + file map**: Keeps the architecture document focused on runtime behavior. File map is a separate reference doc.
- **Old scripts removed, not deprecated**: No transitional wrappers — the rename is complete.
- **Public docs standardized in English**: Kept the public-facing documentation surface consistent with the English-first policy, while leaving historical/internal records in `sdd/` untouched.
